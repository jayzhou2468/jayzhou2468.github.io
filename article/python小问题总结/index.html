引言

最近把python的两本基础书过了过，分别是嵩天编写的《python语言程序设计基础》和小甲鱼编写的《0基础入门学习python》。两本书都很基础，适合初学者看，但是我看完感觉还是有收获的，发现了很多的问题和漏洞。现在我把遇到的问题总结写出来，以后时常温故而知新~

基础知识

  1. 操作符优先级：幂运算>正负号>算术操作符（加减乘除）>比较操作符（<、<=、!=）>逻辑运算符（not、and、or）
  2. 三元操作符格式：
    small = x if x<y else y
  3. range（1,10,2）注意第三个参数
  4. python中可以用列表或者元组return多个值
  5. 局部变量原理：运行函数时，利用栈进行存储，执行完后函数所有数据都会被自动删除，所以在函数外边是无法访问到函数背部的局部变量
  6. lambda表达式可以代替一些简单的函数而不用定义函数。
    如：lambda x, y : x + y
  7. map()函数有点不明白
  8. 生成器yield语句可以暂停或挂起函数，在需要的时候继续执行，使用next(对象)继续执行
  9. 列表推导式：a = [i for i in range(100) if not (i%2) and i % 3]
  10. if name == 'main’作用是只有单独运行这个文件才会执行里面的内容，导入到别的文件时候不会执行

列表、元组、字符串和字典、集合

  1. 向列表添加一个元素用append(1)，向列表末尾添加多个元素用extend([1,2,3])，插入一个元素用insert(num,index)
  2. 切片的用法：
    list[::-1]或list[::2]各自作用
  3. temp = (1)，其类型是int；temp = 1,或者temp = (1,)，其类型是tuple。所以元组只有一个元素的时候记得加逗号
    举个栗子：8 * (8)结果是64，而8 * (8,)结果是(8,8,8,8,8,8,8,8)
  4. join()用法是以字符串作为分隔符，插入到sub字符串所有字符之间
    举个栗子：‘x’.join(“Test”)结果是’Txexsxt’。它的好处是比连接符号“+”拥有更高的效率，用于拼接字符串更好
  5. 字典的定义有多种，一般还是用dict1 = dict(A = 50, B = 39, C = 23)。给字典赋值时，若键存在则覆盖，不存在则创建一个新的键并赋值。格式是dict1[‘x’] = 34
  6. 集合在python的作用就是唯一性，能把重复元素去除，但是它是无序的。创建集合可以直接用大括号，或者用set([1,2,3])
    去重的话就可以用list1 = list(set(list1))实现
  7. 字典和文件支持迭代操作
    如：links = {‘a’:‘www’,‘b’:‘aaa’,‘c’:‘ggg’}
    for each in links:
    print(’%s -> %s’ % (each, links[each]))

文件

  1. f.seek(offset, from)：from为0代表文件起始，1代表当前位置，2代表末尾；将文件指针设置为起始用seek(0,0)即可
    f.read(5)即读取五个字节
    f.readline()从指针开始位置向后读取一整行，遇到“\n”结束
  2. 使用pickle可以将列表、字典这类复杂的数据类型存储为文件

类和对象

  1. super函数用法
  2. 多重继承：class C(base1, base2): pass
  3. 组合：将乌龟和鱼组合成水池类
  4. 注意：属性的名字和方法名相同，属性会覆盖方法
  5. 类的定义要“少食多餐”，不要在一个类里面定义所有特性和方法，而用继承和组合机制进行扩展
